# 사용자 어플리케이션 디자인 가이드 (User Application Design)

본 문서는 앱의 시각적 완성도와 사용자 경험(UX)을 극대화하기 위해 최신(2025~2026년 기준) 프리미엄 음악 앱의 트렌드를 반영한 디자인 가이드라인입니다.

---

## 1. 참고 UI/UX 레퍼런스 및 최신 트렌드

최근 글로벌 음악 스트리밍 앱(Apple Music, Spotify, YouTube Music)의 핵심 디자인 트렌드를 채택하여 **몰입감**과 **역동성**을 강조합니다.

1. **글래스모피즘 (Glassmorphism) 기반의 배경** (참고: Apple Music)
   * 앨범 아트워크를 화면 전체의 배경으로 확대한 뒤, 매우 강력한 **블러(Blur)** 처리를 하여 깊이감을 부여합니다. 음악이 재생됨에 따라 배경의 색상이 은은하게 일렁이는 효과를 줍니다.
2. **동적 컬러(Dynamic Color) 반영** (참고: Spotify)
   * 고정된 테마 색상을 사용하지 않고, 현재 재생 중인 **곡의 앨범 아트에서 주요 색상(Dominant Color)을 추출**하여 플레이어의 배경 그라데이션, '재생' 버튼, '좋아요' 아이콘 등의 포인트 색상으로 실시간 반영합니다.
3. **플로팅 미니 플레이어 (Floating Mini Player)** (참고: YouTube Music, Bugs)
   * 유저가 검색을 하거나 차트를 둘러보는 중에도, 화면 하단(네비게이션 바 바로 위)에 미니 플레이어가 떠 있도록 설계하여 끊김 없는 음악 제어 경험을 제공합니다.
4. **마이크로 인터랙션 (Micro-interactions)**
   * 곡을 재생목록에 담거나 좋아요를 눌렀을 때 튀어오르는 파티클(Lottie) 애니메이션, 미니 플레이어에서 앨범 아트를 탭할 때 전체 화면으로 부드럽게 팽창하는 영웅(Hero) 애니메이션을 적용해 조작의 즐거움을 더합니다.

---

## 2. 컬러 컨셉 (Color Concept)

몰입도를 최고조로 끌어올리기 위해 앱 전체를 **다크 테마(Dark Theme)** 전용으로 설계합니다.

* **Background (배경색)**: `#0D0D0D` (완전한 블랙에 가까운 다크 그레이)
* **Surface (카드, 미니 플레이어 등 표면색)**: `#1A1A1A` (배경보다 약간 밝아 입체감을 줌)
* **Text (텍스트)**
  * **Primary (제목, 강조 텍스트)**: `#FFFFFF` (순백색)
  * **Secondary (아티스트 이름, 부가 설명)**: `#B3B3B3` (가독성이 좋은 연회색)
* **Accent (포인트 색상)**: **동적(Dynamic)**
  * 앨범 아트에 따라 실시간으로 변하는 유동적인 색상을 사용.
* **Semantic Colors (피드백/상태 색상)**
  * 좋아요/활성화: `#FF3B30` (강렬한 레드 포인트) 또는 추출된 Accent Color
  * 에러/알림: `#FF4B4B`

---

## 3. 디자인/애니메이션 구현을 위한 필수 Flutter 라이브러리

화려하고 트렌디한 디자인을 성능 저하 없이 구현하기 위해 아래의 검증된 패키지들을 활용합니다.

| 패키지명 | 활용 목적 및 역할 |
| :--- | :--- |
| **`palette_generator`** | 앨범 아트(이미지)에서 지배적인 색상(Dominant)과 활기찬 색상(Vibrant)을 동적으로 추출하여 앱 배경 및 버튼 색상에 실시간으로 적용합니다. |
| **`cached_network_image`** | 유튜브 썸네일을 불러올 때 이미지 캐싱을 처리하며, 로딩 중에는 자연스러운 페이드인(Fade-in) 효과나 스켈레톤(Shimmer) 애니메이션을 지원합니다. |
| **`flutter_animate`** | 검색 결과 리스트가 뜰 때 차례대로 슬라이드 업(Slide-up)되거나 페이드인되는 연속적이고 세련된 UI 트랜지션 애니메이션을 아주 쉽게 구현합니다. |
| **`animations`** | 하단 미니 플레이어를 터치했을 때, 전체 화면 플레이어로 매끄럽게 변환되는 `OpenContainer` (구글 Material 모션) 트랜지션을 제공합니다. |
| **`lottie`** / **`rive`** | '좋아요(하트)' 클릭 시 터지는 파티클 효과, 음악 재생 중 움직이는 이퀄라이저 파동 등 고해상도의 마이크로 애니메이션을 가볍게 재생합니다. |
| **`glassmorphism`** <br>*(또는 내장 `BackdropFilter`)* | Apple Music 스타일의 뒷배경 블러 처리 및 반투명한 유리 질감의 플레이어 컨테이너 뷰를 설계할 때 사용합니다. |
